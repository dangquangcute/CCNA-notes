1.1 Tính từ có tiền tố / hậu tố 
+ Hậu tố : 
 - able,ible : comfortable , invisible 
 - al,ial : normal, musicial, industrial, social
 - ful : beautiful, harmful, peaceful, wonderful
 - ic : classic, economic, fantastic, romantic
 - ish : British, childish, foolish 
 - ive,ative : active, alternative, creative, talkative.
 - less : endless, motionless, priceless, timeless.
 - eous, ious, ous : spontaneous, ambitious, anxious, dangerous, famous.
 - y : angry, busy, weathlthy, windy.

 + Tiền tố : 
 - un : fair -> unfair, happy -> unhappy, sure -> unsure
 - in : active -> inactive, appropriate -> inappropriate, complete -> incomplete
 - ir : responsible -> irresponsible, regular -> irregular, reducable -> irreducable
 - im : balance -> imbalance, polite -> impolite, possible -> impossible.
 - il : legal -> illegal, legible -> illegible, logical -> illogical.

1.3 Tính từ ghép
 - adv + ed : well-behaved
 - adv + ing : fast-growing
 - adj + ed : ready-made, hand-made, soft-spoken
 - adj + ing : easy-going, long-lasting, good-looking
 - N + ed : action-packed, Hanoi-based
 - ed + particle : made-up 
 Có những tính từ ghép khác k có dạng phân từ : short-term/long-term , small-scale
